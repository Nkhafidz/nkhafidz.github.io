Catatan Penting

Jangan unggah tautan sensitif ke GitHub Pages publik.

GitHub Pages bersifat statis dan pada umumnya dapat diakses publik. Untuk tautan register pasien, SIMRS internal, RME, atau dokumen yang berisi data pribadi, gunakan akses terbatas: VPN, Google Drive restricted access, repository private sesuai paket, atau proteksi tambahan seperti Cloudflare Access.
